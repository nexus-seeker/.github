Kawula is an AI-native operating system for Solana Mobile. We bridge the gap between human intent and secure blockchain execution.

- **[Kawula Mobile](https://github.com/nexus-seeker/kawula-mobile)**: The flagship AI Agent OS for Solana Seeker.

- **[Kawula Programs](https://github.com/nexus-seeker/kawula-anchor-programs)**: On-chain policy guards and secure execution layers.
